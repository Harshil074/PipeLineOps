What blocks a deploy

Secrets in the diff

Any high-entropy string that looks like a credential fails the run immediately, regardless of severity elsewhere.

High or critical CVEs

New dependencies with known high/critical vulnerabilities block the merge; moderate and low findings are logged but don't block.

Lint errors, not warnings

Structural HTML/CSS errors fail the build; style warnings are surfaced in the run log but allowed through, so the bar stays useful instead of noisy.
